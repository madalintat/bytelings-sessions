# Phase 1 — Python Core Fluency

**Days 1–31** (28 module days + 3 phase-project days)

## What you'll be able to do by the end

- Read and write idiomatic Python without reaching for syntax references
- Understand truthiness, identity vs equality, and mutation pitfalls
- Use lists, dicts, sets effectively and know which fits a problem
- Write functions with appropriate `*args/**kwargs`, defaults, closures
- Apply decorators where they actually help (and skip them where they don't)
- Annotate functions with surface-level type hints
- Reason about Big-O for simple operations on lists and dicts

## Modules

| # | Module | Days | What |
|---|---|---|---|
| 1 | Setup + values, types, identity | 1–5 | Your environment, identity vs equality, truthiness |
| 2 | Strings deep | 6–11 | Slicing, methods, encoding, regex, parsing |
| 3 | Lists + linear search + Big-O intro | 12–17 | List ops, iteration, counting operations |
| 4 | Dicts, sets + hashing | 18–22 | Lookup model, why O(1), `Counter`/`defaultdict` |
| 5 | Functions + closures + decorators + type hints | 23–28 | Real function-writing fluency |
